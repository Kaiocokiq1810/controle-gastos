# 💰 Controle de Gastos (CLI)

##  Descrição
Este projeto é uma aplicação simples em Python que permite ao usuário controlar seus gastos pessoais através do terminal (CLI).

##  Problema
Muitas pessoas têm dificuldade em organizar seus gastos diários, o que pode levar a descontrole financeiro.

##  Solução
A aplicação permite registrar, listar, calcular e remover gastos de forma simples e rápida.

##  Público-alvo
Pessoas que desejam organizar suas finanças de forma prática.

##  Funcionalidades
- Adicionar gasto
- Listar gastos
- Ver total
- Remover gasto

##  Tecnologias
- Python
- Pytest
- Flake8
- GitHub Actions

python src/app.py
