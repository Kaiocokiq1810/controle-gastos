# 💰 Controle de Gastos (CLI)

## Descrição

Este projeto é uma aplicação simples desenvolvida em Python que permite ao usuário controlar seus gastos pessoais através do terminal (CLI).

O sistema foi criado com o objetivo de auxiliar no gerenciamento financeiro pessoal, oferecendo funcionalidades básicas para cadastro e acompanhamento de despesas.

## Problema

Muitas pessoas têm dificuldade em organizar seus gastos diários, o que pode levar a um descontrole financeiro e dificultar o planejamento do orçamento pessoal.

## Solução

A aplicação permite registrar, listar, calcular e remover gastos de forma simples e rápida através de uma interface de linha de comando.

## Público-alvo

Pessoas que desejam organizar suas finanças de forma prática e objetiva.

## Funcionalidades

* Adicionar gasto
* Listar gastos
* Ver total de gastos
* Remover gasto
* Consultar cotação do dólar em tempo real através de API externa

## Tecnologias Utilizadas

* Flake8
